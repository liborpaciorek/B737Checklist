Boeing 737-800 Full Flight Checklist APU Only, Avionics, ATC, FMC Steps

1. Preliminary Cockpit Preparation (Cold and Dark)

  • Battery ON, Standby Power AUTO
  • Emergency Exit Lights ARMED
  • Parking Brake SET
  • APU Fire Switch IN
  • APU START (observe EGT/RPM on Upper DU)
  • APU GEN ON (check ELEC synoptic)
  • IRS Mode Selectors OFF then NAV (alignment begins)
  • Oxygen, Documents, Manuals checked
Avionics: ELEC synoptic: APU powering buses; IRS Panel/Upper DU: "ALIGN"/countdown; Standby attitude/clock. PFD/ND blank until IRS completes align.
Obtain ATIS via COMM radio, record QNH/runways/ATIS code.

2. FMC (CDU) Initialization — Before Clearance

FMC/CDU:
  • IDENT Page: confirm database
  • POS INIT: set REF AIRPORT, GPS current/gate position for SET IRS POS
  • ROUTE: ORIGIN/DEST ICAO, flight number, planned route
CDU displays POS INIT/ROUTE; ND displays airport/plan on align complete.
None — prepare for clearance request.

3. ATC Clearance

  • Request clearance from Delivery: e.g. “Clearance Delivery, American 123, stand 14, request IFR to Chicago O’Hare, information Alpha.”
  • Read back clearance (route, squawk, initial altitude)
  • Set assigned squawk in XPDR (STBY)
FMC/CDU:
  • DEP/ARR: Choose DEPARTURE runway and SID as cleared
  • ROUTE: Airway transitions/waypoints
  • LEGS: Clean up discontinuities

4. Before Pushback & Start

  • Doors closed/locked, seatbelt ON
  • Packs OFF, ISOL valve OPEN, APU BLEED ON
  • Beacon ON
  • Transponder code & STBY
FMC/CDU:
  • ACTIVATE and EXEC route if needed
  • PERF INIT: CRZ ALT, COST INDEX, ZFW, RESERVES, GCW
  • N1 LIMIT: FLEX TEMP, N1 limit
  • TAKEOFF REF: Flaps, CG/TRIM, V1/VR/V2 (accept/supplement as needed)
CDU: DEP/ARR, EXECed, PERF, N1, TAKEOFF REF pages; ND: Runway, plan view, IRS complete.
“Ground, American 123, request push and engine start, information Alpha.”
Confirm push direction with ground crew.

5. Engine Start

  • Packs OFF, ISOL valve OPEN, APU BLEED ON
  • Engine 2 start: monitor N2, idle fuel at 25% N2
  • Engine 1 start: same as above
  • Engines stabilized; GEN1/GEN2 ON (ELEC transfer)
  • APU BLEED OFF, ISOL AUTO, Packs ON
FMC: LEGS: first waypoint/runway cue, TAKEOFF REF: V-speeds visible
Upper DU: engines/GENs/pressures green, ELEC: bus transfer to gens; ND: ready.
Notify ready for taxi if required.

6. Taxi & Before Takeoff

  • Brakes/steering test; recall check
  • Flaps set for takeoff (per FMC)
  • MCP: SPD V2, HDG, ALT match FMC/MCP, FDs ON, A/T ARM ON
  • Flight instruments crosschecked
  • Lights: taxi, strobes, landing
  • Cabin crew notified for takeoff
TAKEOFF REF: N1/V-speeds final check; LEGS: runway active
Lower DU: Flaps/trim, spoilers ARM; ND: taxi/plan/map, magenta route
“Ground, American 123, request taxi, gate 14, Alpha for Runway 22R.”
“Tower, American 123, holding short runway 22R, ready for departure.”

7. Takeoff

  • Line up; thrust set; all lights ON
  • Calls: 80 knots, V1, VR, positive rate—gear UP
TAKEOFF REF: Review; LEGS: runway to SID leg
PFD: TO/GA, ASI, V1/VR/V2 bug; ND: SID sequence
“Cleared for takeoff runway 22R, American 123.”
On rotation: “Contact Departure…”

8. After Takeoff & Climb

  • Flaps up on schedule
  • Engine start switches OFF at 1500ft AGL
  • Packs ON, APU OFF at safe alt
LEGS: Sequence, next restriction; PROG: ETA, next waypoint
ND: SID/route, PFD: LNAV/VNAV active, FMA/ALT/SPEED
“Departure, American 123, passing 1,800 climbing 5,000.”

9. Climb/Cruise

  • Monitor engines, fuel, pressurization
  • MCP/AUTO modes follow climb/cruise
PROG: Monitor cruise ETA/fuel; LEGS: track route, make directs if needed
PFD: CRZ, ND: magenta route, CDU PROG active
“Center, American 123, FL350, level.”

10. Pre-Descent/Descent

  • ATIS received, QNH set
  • Approach briefing, set STAR/approach in FMC
  • MCP ALT set to cleared level
  • Seatbelts/landing lights as needed
DEP/ARR: Select ARRIVAL, STAR, transition; LEGS: verify fix/speeds/alt; PROG: EFOB/fuel/ETA
ND: STAR/approach profile; PFD: Alt/vertical mode
“American 123, descend and maintain FL240, cross BAF at 12,000, QNH 1015.”

11. Descent

  • MCP ALT set to cleared constraint
  • QNH set at transition, speed as cleared
  • Seatbelts/approach checklist
  • ILS/APP NAV aids set
LEGS: Constraints monitored and cleaned; PROG: Arrival/fuel
PFD: ALT tape, "VNAV PATH" or "ALT" mode; ND: approach track/plan
Acknowledge further descent/approach clearances.

12. Approach

  • Tune/set flying ILS or visual (NAV RAD/APP NAV)
  • Final flap setting, gear DOWN @ 2,000-1,500' AGL
  • MCP set for final approach
  • Landing checklist complete
LEGS: Final approach fix, minima; PROG: Final checks, fuel/ETA
PFD: FMA GS/LOC (ARM/CAPT), VREF bug; ND: Approach track; CDU LEGS active
“Tower, American 123, established ILS 22R.”
“Cleared to land runway 22R, American 123.”

13. Landing

  • Gear DOWN, three green
  • Flaps FULL, Spoilers ARMED
  • Landing checklist complete
  • Touchdown with reverse/thrust as briefed
PFD: GS/LOC centered, VREF check; ND: Final DME; Lower DU: FLAP & GEAR green
“Exit when able, contact Ground…”

14. After Landing

  • Spoilers DOWN
  • Flaps UP
  • APU START
  • Flight Directors OFF, XPDR STANDBY
  • Landing/Strobe lights OFF, Taxi ON
Lower DU: Flap/trim/spoilers; Upper DU: APU EGT rising; CDU: next leg if turnaround
“Ground, American 123, clear of 22R, taxi to gate 14 via Bravo.”

15. Shutdown

  • Parking brake SET
  • Engine levers CUTOFF, APU GEN ON
  • IRS OFF, Fuel pumps/hyds/probe/window heat OFF
  • XPDR OFF, Emergency Lights OFF
  • Battery OFF (last)
Route delete; pages go blank after IRS off
ELEC: APU supply, blank after battery off
Confirm parking/shutdown position if required
